iOS Repository

The Tor for iOS repository has been discontinued: jailbreaking your device in order to install additional software lowers its security barriers and expands the attack surface.

For an alternative solution, have a look at the Onion Browser in the App Store.

If you are interested in past efforts, have a look at the code repository. It is an overlay for Saurik's Telesphoreo Tangelo repository and the debs have been built using this Linux cross compiling toolchain.
